# 📊 Customer Churn Prediction Pipeline

An end-to-end **Machine Learning project** that predicts customer churn using data preprocessing, model building, and evaluation techniques.
This project demonstrates how to build a complete pipeline from raw data to predictive insights.

---

## 📌 Project Overview

Customer churn prediction helps businesses identify customers who are likely to stop using their services.
In this project, we:

* Analyze customer behavior
* Perform data preprocessing
* Build machine learning models
* Predict churn probability

---

## 📂 Dataset

* Customer dataset containing user details and service usage

### Features Included:

* Customer ID
* Gender
* Age
* Tenure
* Services Used
* Monthly Charges
* Total Charges
* Contract Type
* Churn (Target Variable)

## 🔍 Project Workflow

### 1. Data Collection

* Loaded dataset
* Inspected structure and values

### 2. Data Preprocessing

* Handled missing values
* Encoding categorical variables
* Feature scaling

### 3. Exploratory Data Analysis (EDA)

* Distribution analysis
* Feature relationships
* Correlation analysis

### 4. Model Building

* Logistic Regression
* Decision Tree
* Random Forest

### 5. Model Evaluation

* Accuracy Score
* Confusion Matrix
* Classification Report

## 💡 Key Insights

* 📉 Customers with higher monthly charges are more likely to churn
* 📆 Short tenure customers have higher churn rates
* 📄 Contract type plays a major role in churn prediction
* 👥 Certain customer segments are more likely to leave

